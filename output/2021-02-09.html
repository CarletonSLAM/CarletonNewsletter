Introduction to Scientific Python

In the introduction to the scientific python workshop, we will be delving into some of the most common data science tools that are used such as Numpy, Pandas, Sci-Kit Learn and matplotlib. The goal would be to showcase some of the common operations we use in standard data science/ machine learning workflows and to share some of the intuition behind the inner working of these libraries. This workshop is for those with a beginner understanding of Python.
