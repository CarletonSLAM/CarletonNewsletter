IEEE and SCESoc GitHub Workshop
Date: Tuesday September 24, 2019
Time: 6:00 PM - 8:00 PM
Location: CB 5109

Have you ever wanted to program on a project with a team? Have you wanted to learn how industry and open-source collaborate in coding around the globe? Come learn how to use Git with SCESoc and IEEE! This workshop comprises of both learning simple Git theory and practical application!

The workshop starts at 6 PM in Canal Building 5109. This lab is booked to provide a preset environment for the participants to focus on learning and using the tool.However, if you are comfortable with the command line and have GIT set up on your computer, feel free to bring it.
